Sheet 002 — Project Detail

Fit Tool, Version N

Version N of the Fit Tool serves the same purpose of the original -- allows potential clients to measure their horse to find the right saddle, the right fit. Version N removes the need for separate parts that could be lost.

Material PLA
Process 3-D Printed
Status Complete

This interactive 3D model needs a browser that supports 3D content — try a recent version of Chrome, Edge, Firefox, or Safari.

Interactive model — drag to rotate, scroll to zoom.

Design Brief

The tool needed to serve the same purpose of the original design without increasing material cost or print time.

Approach

Version N simplifies the original design of V.1 and the built in connectors of V.2 recreating the parts from scratch while referencing the critical dimensions of the original.

Outcome

Version N removes the need for small fastener parts make it easier for the producer to assemble and easier for the end customer to use.
