Drawer-Mounted Remote Control Holders

A set of custom holders made for four different styles of remote control, giving loose remotes a fixed home on the front of a drawer.

Material PLA
Process 3-D Printed
Status Complete

This interactive 3D model needs a browser that supports 3D content — try a recent version of Chrome, Edge, Firefox, or Safari.

Interactive model — drag to rotate, scroll to zoom.

Design Brief

Remotes had a tendency to become misplaced or lost between couch cushions without a convenient place to store them. Each of the four remotes in use is a different size, so a single generic holder wouldn't fit them all.

Approach

The drawer the holders hook onto and each remote were measured for length, width, and depth, with clearance left around the buttons so nothing is pressed while a remote sits in its holder. Each holder was shaped to its specific remote and given a hook that clips over the drawer edge.

Outcome

A convenient storage solution for loose remotes that needs no modification to the drawer the holders hook onto — the holders simply lift off if the drawer is ever used for something else.
